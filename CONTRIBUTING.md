# Contributing

Thanks for considering contributing to hermes-devops-skills!

## How to add a skill

1. Fork the repo
2. Create `skills/your-skill-name/SKILL.md` with YAML frontmatter:
   ```yaml
   name: your-skill-name
   description: Clear one-line description of what this skill does
   author: Your Name
   ```
3. Describe the DevOps pain point, solution, and step-by-step instructions
4. Open a pull request

## Guidelines

- One skill per directory
- Keep descriptions English (global audience)
- No hardcoded local paths (`/Users/...`)
- Include a "When to use" trigger description
